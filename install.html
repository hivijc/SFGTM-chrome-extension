Troubleshooting

Chrome shows "Developer mode extensions" warning on startup

This is normal. Just click the X to dismiss it. It appears because the extension wasn't installed from the Chrome Web Store.

I don't see the sidebar on LinkedIn

Make sure you're on a linkedin.com/in/ or linkedin.com/company/ page. Try refreshing the page. Check that the extension is enabled at chrome://extensions.

How do I update the extension?

Download the latest ZIP from GitHub again, replace the old chrome-extension folder, then go to chrome://extensions and click the ↻ refresh icon on the Nat-vigator card.

The extension shows "Network error"

This usually means the backend server is down or your internet is disconnected. Try again in a few minutes. If it persists, contact Nathaniel.

Can I move or rename the folder after loading?

No. Chrome references the original folder. If you move or delete it, the extension will break. Keep the extracted folder in a permanent location (e.g., your Desktop or Documents).
